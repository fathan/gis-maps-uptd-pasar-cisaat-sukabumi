# Database Configuration
- page/page_config.php
``` bash
<?php
	ini_set('display_errors',FALSE);

	$host		="localhost";
	$user		="root";
	$pass		="fathan123";
	$db 		="project_gis_pasarcisaat";

	$koneksi=mysql_connect($host,$user,$pass);
	mysql_select_db($db,$koneksi);	
?>
```
